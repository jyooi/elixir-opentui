# TuiTodo

An interactive terminal todo list app built with [ElixirOpentui](https://github.com/jyooi/elixir-opentui).

## Running

```bash
cd example_app/tui_todo
mix deps.get
mix run todo.exs
```

## Controls

- **Tab / Shift+Tab**: Navigate between input, filter, and todo list
- **Enter**: Add a new todo
- **Space**: Toggle todo completion
- **d / Delete**: Delete a todo
- **Left / Right**: Switch filter (All / Active / Done)
- **Up / Down**: Navigate todo items
- **Ctrl+C**: Quit
